# Mobile-Sales-Analysis-Power-BI-Project
In this project, I analyzed mobile phone sales data to uncover business insights, identify top-performing brands, and understand regional and pricing trends.
## 🔍 Project Objectives
- Analyze total mobile sales, quantity sold, and brand performance.
- Visualize trends across regions and pricing categories.
- Identify best-selling mobile models.
- Enable dynamic filtering and interactive exploration.

## 📌 Key Features
- **KPI Cards** for Total Sales, Units Sold, and Average Price.
- **Bar Charts** showing brand-wise and model-wise performance.
- **Line Charts** for time-series sales trends.
- **Slicers** for filtering by brand, region, price, and model.
- **Pie Charts** for market share visualization.

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Cleaning
